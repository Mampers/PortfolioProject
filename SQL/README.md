# SQL Projects

This folder contains SQL-based projects demonstrating data cleaning, transformation, and analytical querying using relational databases (primarily SQL Server).

The focus is on structured querying, dataset transformation, and analytical logic.

---

## Projects in This Folder

### 1) COVID-19 Data Exploration
**File:** `COVID_19.sql`

Exploratory analysis of COVID-19 data using SQL.

Key techniques:
- Aggregations (SUM, COUNT, AVG)
- GROUP BY analysis
- Filtering and conditional logic
- Trend-style queries
- Derived metrics

Skills demonstrated:
- Analytical querying
- Data summarisation
- Business-style metric calculations
