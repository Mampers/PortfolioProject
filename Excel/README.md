# Excel Projects

This folder contains Excel-based analytics projects demonstrating strong fundamentals in data cleaning, pivot analysis, and dashboard-style reporting.

## Projects

### 1) Power BI Final Project (Excel Source)
**File:** `Power_BI_Final_Project.xlsx`  
**Purpose:** A prepared Excel dataset/workbook used as a source for a Power BI reporting project.  
**Skills demonstrated:**
- Data cleaning and structuring for analysis
- Consistent column naming and tidy-table layout
- Basic Excel modelling to support downstream BI

**How to use:**
1. Download the file.
2. Open in Excel.
3. Review the data structure and any supporting tabs (if present).
4. Use as an input to Power BI (Get Data → Excel).

---

### 2) Pivot Table Bank Account Project
**File:** `Pivot_Table_Bank_Account_Project.xlsx`  
**Purpose:** Pivot-table driven analysis of a banking-style dataset to summarise and explore trends quickly.  
**Skills demonstrated:**
- Pivot Tables / Pivot Charts
- Sorting, filtering, grouping
- KPI-style summaries and quick insights

**How to use:**
1. Download the file.
2. Open in Excel.
3. Refresh pivots (Data → Refresh All) if needed.
4. Review pivot fields and filters to understand the logic.

## Notes
- Excel files are included primarily to show analysis fundamentals and clean input preparation for BI tools.
- Where applicable, corresponding Power BI reports may exist in the `Power_BI/` folder.
